## Installation Guide 📥

### Standard Installation (Windows / macOS)

1. Download the installer via the official Adobe Creative Cloud desktop application
2. Run the setup file and follow the installation instructions
3. Log in with your Adobe account credentials
4. Launch Adobe Premiere Pro from your desktop or application menu

### Installing Motion Graphics Templates (.mogrt)

1. Open the **Essential Graphics** panel inside Premiere Pro
2. Click the **Install Motion Graphics Template** button in the lower corner
3. Select your `.mogrt` file to add custom titles or lower-thirds to your project
